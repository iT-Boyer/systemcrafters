Welcome!

System Crafters 是一个专门为那些有兴趣通过 增加使用乐趣和生产力 来设计自己的计算体验的计算机爱好者学习和资源社区。

这个网站旨在成为互联网上学习如何利用 GNU Emacs、GNU Guix 和 GNU/Linux 等工具来使用自由软件工具完全定制工作流程的最全面参考资料!

System Crafters YouTube 频道 也包含近 200 个视频和现场流录制,演示和解释这些工具和更广泛的主题!

¶开始!

如果您刚开始使用 GNU Emacs 并想了解基础知识,请查看 Emacs 绝对初学者指南!

从那里开始,查看 Emacs 要点 系列以了解使用 Emacs 的基础知识,然后按照 Emacs 白板 系列学习如何构建一个您理解的现代 Emacs 配置!

一旦您熟悉使用 Emacs,请通过查看 用 GNU Guix 打造你的系统 系列学习如何使用 GNU Guix 来制作完整的、可重现的系统配置!

您还可以查看我创建的许多 视频系列 ,以帮助您深入学习这些工具。
